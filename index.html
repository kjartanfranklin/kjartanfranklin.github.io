How to make a website

Project description
The goal of this project was to make a website using HTLM and then to upload it to Github so it could be accessed through any browser. The website will be used throughout the remainder of this course to hand in and store assignments. The website also has the potential to be useful after I graduate, e.g. when I apply for a post-graduate education or apply for a job.

First things first
To make a website using HTML you'll need the right software to get the job done. So the first thing I did was setting up Brackets so I could write the HTML script and test the result. One feature that I really liked, and used a lot, was the live view button. When you press the live view button (as demonstrated below) you can see how your website is coming along.

I had never before worked with HTML so it wasn't realistic to write the whole script on my own. I needed some help so I browsed through templates made by HTML5 UP. I really enjoyed the minimalistic features of the template called Dimensions so I downloaded it, which is free when it's not used for business related matters. At this point I was all set up to make the look of my very own website.

The script
First I opened up the template by HTML5 UP in Brackets. I was able to change the text where I wanted to but when I wanted to change the format and style I needed to gather additional information. w3schools.com came in really handy for learning more about HTML. With their help I found out how to change images, text styles, how to insert linkes and much more!
It was also important to read the script that came with the template and try to understand what was going on. When you understand the script you can use it to your advantage. I also looked at the websites of two former students for inspiration, but I tried to make the website as much my own as I could. The image below demonstrates how w3schools.com helped me to align the image on the home page so it would fit into the circle.

Uploading to Github
I already had a Github account but if you don't have one you can create it here. The next step was to make a new repository, have in mind that the name of the repository will be the name of the website. Next I scoured the Internet for information on how to upload my website to my repository on Github. I spent a few hours trying to figure it out and asked my fellow classmates for help but there seemed to be a problem going on nobody quite understood. Down below you can see one of the errors I delt with.

After many trials and errors on the Terminal application on my MacBook I think a found the simplest way of uploading the website into my repository. Here it is step by step:

1. Open up your repository.

2. Select add file.

3. Select upload files.

4. Choose your files or drag them into the box.

5. Wait for the files to load.

6. Press Commit changes.

That's it! After that I was able to go to kjartanfranklin.github.io in my browser and enjoy my new website.

To be fair, I am unforunately not entirely sure if some of the things I did in the Terminal application made this process possible.

Looking back
As I said in the column About me I really enjoy projects where there is room for creativity, so I really enjoyed this project. I have often wondered about website design and development and wanted to learn more about it. Now I have some basic knowledge on how to make a website using HTML and I certainly appreciate it.

Press fit laser cut

Project description
The goal of this project was to make a press fit model, with parameters, of a building unit. The model has to be scaleble so it's possible to adjust kerf, thickness, sizes of the surfaces, by working with parametric variables.

Idea
I got a new Nespresso machine this christmas and I'm currently working on making a nice Nespresso station. I had already bought a tray and coffee mugs but I still needed an object to store all the Nespresso capsules. When I read the project description I immediately decided I was going to make something that I needed and was able to get some use out of. With that said I thougt it would be perfect for me to build a unit that stores Nespresso capsules. I also decided to add another compartment to the unit for used capsuled so I don't have to make a trip upstaires to the recycling bag everytime I use a capsule. I thought it would be best to design a slider to remove the compartment for the used capsules.

Design

For the design I used Fusion 360, you can download the application here. I was already familiar with other CAD programs like AutoCad Inventor, so I knew I would be able to make the design in Fusion without to many problems. On the image above you can see how the design looks in Fusion 360. Hafliði, our instructor, linked a YouTube video on how to make finger joints in Fusion 360 wich you can find here. This video helped me out a lot and it also shows how to work with parameters. Parameters are essential if you want to be able to adjust the size of your design with out much trouble. On the image below you can see my parameters in this project.

Adjusting for kerf
Before I had finished my design I went to the Fablab in Breiðholt to take a look at the laser cutting machine with my group. We needed to find out the kerf of the laser so we could adjust our design to the widht of the laser. Kerf is a word used to describe the thickness that the laser cuts off from the surface of the object. The finger joints of the object would not be press fit without adjusting for kerf. You can read all about how to measure kerf here.
When we had measured the kerf I installed a plug in for Fusion 360 called DXF for Laser, you can find it here. DXF for laser is super easy to use. You can follow these steps to adjust the surfaces of your object for kerf and save them as a DXF file using DXF for laser:

1. In Fusion 360 click on create and select "Save DXF for Laser Cutting" from the drop down menu.
2. Select the face of the surface you want to adjust for kerf.
3. Type in the kerf you measured and press "ok".
4. Save the .dxf file in the folder of your choosing.

Getting ready for laser cutting
For the next part you will need to download an application called Inkscape. You can find it here it's free!. Once you've set up Inkscape open up a new file. It's important that you have the page the same size as the material you are cutting from. In my case it was 300x600mm. To change the page size go to File-Document Properties and from there it should be self explanatory.
Now you can open up all the dxf files you saved in the previous step and distribute them so that they all fit on the page. Next you will have to change the line thickness. In my case the laser cutter cuts out all lines with 0,02 mm thickness. To change this I opened the Fill and Stroke menu, made sure there was no fill and changed the widht in the Stroke style colomn. All other lines or shapes that are not 0,02mm are rastered on the material you are using.
I wanted to raster a image on each side of my object to make it look cooler. The way I did it was to find a simple png image online and import it to Inkscape. I had to be able to control the fill and stroke. To do this you have to have the image as an object. Select Path and from the dropdown menu select Trace bitmap. This will not work for complex images. Next I opened up the Fill and Stroke menu, removed the stroke and had only fill. The only thing left to do is to save the Inkscape file as a pdf document. The image below shows my Inkscape document with the Fill and Stroke menu opened.

Laser cutting

Once you have your pdf file you are ready to use the laser printer. To cut out your object using a laser cutter follows the same steps as cutting out the kerf test so I'll let it suffice to refer to the kerf test documentation which you can find here. After cutting out the object all that is left is to put the pieces together. If your object is press fit like mine you will probably have to use some sort of hammer to get the finger joints together (I used a rubber maller), but be carefull not to brake the finger joints.
Thank you for reading about this project and if you decided to follow this process then congratulations on your brand new Nespresso capsule container! Down below you can see a short gif of the design coming to life.

3D printing constraints

Guðmundur Freyr, Gyrðir Hrafn and Kjartan Franklin

Before 3D printing an object with a 3D printer it is important to know the constraints of the 3D printer. The most important constraint we look for is at what angle degree the printer starts "failing" with out support. This is important for us to know so we can adjust the support of our model before 3D printing.
Our instructor, Hafliði, allowed us to use the same model that another group had already made. The only thing we had to do was to set it up in Ultimaker Cura which is a free 3D printing software that you can find here.

The image above showes the settings we used in Cura. We had never before worked with the software so we relied on this video, to get all the information on how the most important settings work. As you can see on the image above we used a layer height of 0.2 with 20% infill. We had no support for obvious reasons and no adhesion.
The next thing to do is to press "Slice" in cura. The image below shows the outcome.

After that our model was ready for 3D printing. It is important that you select the right printer in Cura, in our case we worked with Ultimaker3 Extended. Then we saved the file from cura to a USB drive, plugged it in to the printer and started the 3D printing process.

When the printer had finished its job it was time to take a look at the result. As you can see on the images below the printer started "failing" at around 60 degrees. So we knew that for our individual models the support settings should be: Everything over 60 degrees should have support.
